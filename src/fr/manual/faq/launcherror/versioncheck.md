# 【无法打开】提示“数据查询失败”、“版本查询失败”

这是由于使用用户过多，服务器压力激增导致的数据访问卡死。

- 如果进入地图后筛选器没有点位，可以尝试重启客户端

- 如果进入地图后筛选器项目正常，则无需重启客户端，可以正常使用。
